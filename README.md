# Subin S™ — Creative Designer, UI/UX & AI Visual Creator

A stunning, premium, and highly interactive multi-disciplinary creative portfolio website cloned from the Netlify site.

## 🚀 Key Features

- **Rich Aesthetics**: Premium dark/light theme switching with custom HSL gradient glow spots and interactive glassmorphism.
- **Dynamic Projects Grid**: Responsive filter grid showing high-resolution WebP project mocks.
- **Horizontal Scroll Carousel**: Smooth interactive horizontal section transitions for horizontal services track cards.
- **Generative AI Showreel**: Integrates MP4 video presentations showcasing AI visual creation workflows.
- **Sticky About Section**: Side-sticky active-state image sliders responding dynamically to layout scroll offsets.
- **Interactive Forms**: Responsive contact forms with local state validation and custom fields.
